What is this?

It's a set of simple text-manipulation tools which I wrote to help me with my work. I'm sharing them here so they can help you too.

Is this secure?

All the code to transform your text is run client-side (on you computer) and no data is transmitted to any external servers (it stays on your computer) so it's as safe and secure as the maching you're running it on.
