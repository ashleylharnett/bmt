Build My Text beta

What's this?

This is a tool to help build retrieval strings for use in the SITS, it is primarily aimed at administrators but has some other features which might be useful for SITS developers.

The Retrieval Strings tab generates a retrieval string based on a list of data pasted into the first text box. The operand option lets you set what delimiters should be used in the string. You can also choose to trim the salsh numbers (so 123456/7 becomes 123456).

The Condition Builder tab generates a conditional text statement based on a comparison between a field and a list of data using the "new style" SRL sytax.

The Search & Replace List tab generates &S search and replace syntax from a 2-column list of values.

  • Retrieval Strings
  • Condition Builder
  • Search & Replace List

Note that SITS will usually only retrieve a maximum of around 400 records at a time in this way. Select your options and paste a column of data in the field below to start.
